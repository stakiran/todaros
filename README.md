# todaros
Create your today daily routines. 

## Requirements
- Python 3.7+

## How to use
- Write your routines to each files.
    - @1
    - @2
    - @3
    - DOWs
    - Days
- Everyday, do belows
    - 1: Do `python todaros.py`.
    - 2: Open `daily.md`.
